# Игра «Розпуста» — онлайн MVP

Настольная игра 18+ для пар, веб-версия с генерацией карточек через AI.

## Запуск

1. Открой `config.js` и вставь свои API-ключи (см. инструкции внутри файла).
2. Открой `index.html` в браузере (двойной клик).
3. В Chrome/Edge: меню → «Установить приложение» — будет иконка на рабочем столе.

## Стек

- Чистый HTML + JS (без сборки)
- Anthropic Claude — генерация заданий
- Google Gemini (NanoBanana) — генерация фонов и иконок
- PWA — установка на рабочий стол

## Файлы

- `config.js` — твои API-ключи (НЕ коммить)
- `index.html` — точка входа
- `app.js` — логика игры
- `styles.css` — оформление
- `cards.json` — база предзагруженных карточек
- `manifest.json` — PWA манифест
- `assets/` — сгенерированные картинки (фоны, иконки)
- `фото/` — оригинальные фотки коробки и карточек
